# 🤖 Crypto Signal Bot — Инструкция по установке

## Что делает бот
Каждые N часов собирает данные из 8 источников, анализирует через Claude AI и присылает торговый сигнал в Telegram.

**Источники данных:**
- CoinGecko — цена, объём, рыночная капитализация
- Binance — фандинг рейт, технические данные (SMA)
- Fear & Greed Index — настроение рынка
- Reddit r/Bitcoin — настроение сообщества
- CryptoPanic — горячие новости
- Coinglass — ликвидации (опционально)

---

## ШАГ 1 — Создай Telegram бота (5 минут)

1. Открой Telegram, найди **@BotFather**
2. Отправь команду `/newbot`
3. Придумай название (например: `My Crypto Signals`)
4. Придумай username (например: `my_crypto_signal_bot`)
5. BotFather пришлёт **токен** вида `1234567890:AAxxxx...` — сохрани его

**Узнай свой Chat ID:**
1. Найди в Telegram **@userinfobot**
2. Нажми `/start`
3. Бот пришлёт твой **ID** (число) — сохрани его

---

## ШАГ 2 — Получи API ключ Claude (5 минут)

1. Зайди на **https://console.anthropic.com**
2. Зарегистрируйся (или войди)
3. Слева: **API Keys** → **Create Key**
4. Скопируй ключ вида `sk-ant-xxxx...` — сохрани

**Стоимость:** примерно $15-25 в месяц при запуске каждые 6 часов

---

## ШАГ 3 — CryptoPanic (опционально, бесплатно)

1. Зайди на **https://cryptopanic.com**
2. Зарегистрируйся → Profile → API Access
3. Скопируй **Auth Token** — сохрани

---

## ШАГ 4 — Деплой на Railway.app (10 минут)

Railway — это хостинг где бот будет работать 24/7.

### 4.1 Подготовь файлы
У тебя должны быть три файла в одной папке:
```
crypto_bot.py
requirements.txt
.env
```

Создай файл `.env` (скопируй из `.env.example`) и заполни своими ключами.

### 4.2 Загрузи на GitHub
1. Зайди на **https://github.com** и зарегистрируйся
2. Нажми **New repository** → назови `crypto-bot` → **Create**
3. Нажми **uploading an existing file**
4. Перетащи свои три файла → **Commit changes**

### 4.3 Задеплой на Railway
1. Зайди на **https://railway.app**
2. Войди через GitHub аккаунт
3. Нажми **New Project** → **Deploy from GitHub repo**
4. Выбери `crypto-bot`
5. Railway автоматически установит зависимости из `requirements.txt`

### 4.4 Добавь переменные окружения
1. В Railway: вкладка **Variables**
2. Добавь каждую переменную из `.env` файла:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
   - `CLAUDE_API_KEY`
   - `CRYPTOPANIC_TOKEN` (если есть)
   - `ANALYSIS_INTERVAL_HOURS` = `6`
3. Нажми **Deploy**

**Готово!** Бот запустится и сразу пришлёт первый анализ.

## Частые вопросы

**Q: Как изменить частоту анализа?**
В Railway → Variables → `ANALYSIS_INTERVAL_HOURS` = нужное число (например 4)

**Q: Можно ли добавить другие монеты?**
Да — напиши мне, доработаем код под ETH, SOL и другие

**Q: Бот даёт гарантии прибыли?**
Нет. Бот агрегирует данные и делает вероятностную оценку. Всегда используй собственный анализ и риск-менеджмент.
